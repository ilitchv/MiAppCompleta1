# Project Brief: Beast Reader Lotto

## Visión de Alto Nivel

Construir una aplicación web avanzada para la gestión de apuestas de lotería, utilizando React, TypeScript y Tailwind CSS. La aplicación debe ser rápida, intuitiva y estéticamente agradable, con un tema dual (claro/oscuro) y acentos de neón.

## Metas y Requisitos Clave

1.  **Entrada de Jugadas Flexible:** Permitir la entrada manual de jugadas, así como métodos rápidos como un "Wizard" y un escáner de tickets basado en IA.
2.  **Gestión Completa:** Facilitar la selección de múltiples sorteos (tracks) y fechas, con cálculos de costos automáticos.
3.  **Validación Robusta:** Implementar un sistema de validación para prevenir errores antes de generar un ticket.
4.  **Generación de Tickets:** Producir un ticket final con aspecto profesional, incluyendo un código QR, que sea descargable (PNG) y compartible (JPEG).
5.  **Persistencia de Datos:** Usar `localStorage` para que los usuarios no pierdan su trabajo al recargar la página.
6.  **IA Integrada:** Utilizar la API de Gemini para una funcionalidad de OCR (Reconocimiento Óptico de Caracteres) precisa para escanear tickets físicos.
