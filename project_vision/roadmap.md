
# Project Vision & Roadmap

Este documento describe la visión a largo plazo para **Beast Reader Lotto**.

## Plan de Implementación Modular

### Módulo 1-4 (Completados/En Progreso)
*   Cuentas, Historial, Analíticas Básicas, Notificaciones.

### Módulo 5: Beast Office Command Center (EN CURSO)

-   **Característica 5.1: Gestión de Usuarios** (Completado v5.2)
-   **Característica 5.2: Control Financiero** (Completado v5.2 - Básico)

-   **Característica 5.3: Gestión de Red (Referidos)** (Fase 1 Completada v5.3)
    -   Visualizar árbol de referidos. (Hecho)
    -   Enlaces de invitación y aprobación. (Hecho)
    -   **Fase 2 (Pendiente): Sistema de Compensación.** Integrar motor de cálculo de comisiones basado en el Plan de Compensación (documento externo).

### Módulo 6: Beast Ledger (Economía Segura & Trazabilidad) - PRÓXIMA PRIORIDAD

**Objetivo:** Garantizar que el dinero dentro del ecosistema sea infalsificable y 100% rastreable.

-   **Concepto Central:** Cada unidad de moneda es un "Token" con un Hash Único.
-   **Característica 6.1: Cadena de Custodia (Blockchain-like)**
    -   Rastreo de origen mediante `parentHash`.
    -   Inmutabilidad de transacciones.
-   **Característica 6.2: Panel de Auditoría de Trazabilidad**
    -   Inspección visual del flujo de dinero.

### Módulo 7: Estrategia Premium (Futuro)
-   Generadores de números basados en IA/Estadística.
